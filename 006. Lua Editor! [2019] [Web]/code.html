This can run Lua programs. All you have to do is write your program in the box above and press the “RUN” button to run it. The standard print function as defined by the script that runs the Lua writes to the javascript console, so I had to redefine it. There was also no io.write function. In fact, there was no io library at all! So I had to define io to be a table with the write function as its only element so you could use io.write.

“FIX SPECIAL CHARACTERS” turns smart quotes (‘quote’ “quote”) into regular quotes ('quote' "quote") and dashes (—) into double hyphens (--) and bullets (•) into asterisks (*).

Feel free to copy this code but with print("Hello, World!") replaced with your program to post Lua programs. Just make sure to give me credit!
